# Traveler Context

Use this file when the recommendation depends on the known traveler profile.

| Factor | Detail | Implication |
|--------|--------|-------------|
| Canoe | 16ft aluminum, ~75 lbs | Heavy portages, noisy, wind-sensitive, rugged |
| Portage limit | Under 500m preferred | Rules out deeper carry-heavy routes |
| Open water experience | Some, not extensive | Georgian Bay needs extra caution and island-hopping bias |
| Satellite comms | Garmin InReach Messenger recommended, may not yet be owned | Confirm status before each trip |
| Budget | Lean | Avoid expensive upgrades unless safety-critical |
| Summer plan | Kawartha -> Massasauga -> Algonquin Rain Lake | Sequence matters; build difficulty progressively |
| Motorboat traffic | Strongly dislikes camping on motorboat lakes | Transit through motor areas can be acceptable, camping there is not |
