# Critique Framework

## Review sequence

1. Summarize the core claim or plan.
2. Identify the hidden assumptions.
3. Stress-test the assumptions.
4. Surface the largest blind spot.
5. Give a verdict such as:
- solid
- shaky
- red flag
6. Recommend mitigation.

## Output pattern

Useful sections include:
- assumptions
- strongest counter-arguments
- biggest blind spot
- severity
- mitigation
