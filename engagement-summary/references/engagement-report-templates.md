# Engagement Report Templates

Use these templates when drafting a full report.

## Standard report structure

```text
WHAT WE HEARD: [Project Name]
Stakeholder Engagement Summary

EXECUTIVE SUMMARY
- Engagement overview
- Key themes (3-5 bullets)
- Top recommendations heard

1. ENGAGEMENT OVERVIEW
2. WHO WE HEARD FROM
3. KEY THEMES
4. QUANTITATIVE RESULTS
5. FINDINGS BY STAKEHOLDER GROUP
6. HOW INPUT WILL BE USED
```

## Participation summary template

Include:
- objectives
- methods used
- promotion/outreach methods
- participation totals
- any timeline or geography notes

## Demographics template

Useful dimensions:
- age or life-stage where available
- geography or neighborhood
- rider vs. non-rider status
- stakeholder type

## Theme analysis template

For each theme, capture:
- theme name
- what participants said
- how often it came up
- representative quote
- implications

## Quantitative results template

Use simple tables for:
- option preferences
- satisfaction levels
- barriers
- priorities
- ranking results

## Stakeholder-group analysis

Common groupings:
- current riders
- non-riders
- business community
- institutional partners
- regional partners

For each group, summarize:
- core needs
- biggest concerns
- key requests

## Conflicting input template

```text
Perspective A:
- who held it
- what they said
- why it matters

Perspective B:
- who held it
- what they said
- why it matters

Staff observation:
- neutral interpretation of the trade-off
```

## Closing the loop

End with:
- how input will inform the work
- timeline or next milestones
- how participants can stay informed
