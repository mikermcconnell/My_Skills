# Municipal Council Report Writer

Create professional staff reports following Ontario municipal standards, specifically tailored for City of Barrie format.

## Standard Report Structure

```
STAFF REPORT
To:        [Committee/Council Name]
From:      [Department/Division]
Date:      [Report Date]
Subject:   [Descriptive Title]
File No:   [Reference Number if applicable]

RECOMMENDATION(S):
[Numbered recommendations in formal resolution format]

EXECUTIVE SUMMARY:
[2-3 paragraph summary for busy councillors]

BACKGROUND:
[Context and history]

ANALYSIS:
[Detailed examination of issue/proposal]

FINANCIAL IMPLICATIONS:
[Operating and capital budget impacts]

STRATEGIC ALIGNMENT:
[Connection to Strategic Plan priorities]

STAKEHOLDER CONSULTATION:
[Who was consulted and key feedback]

ALTERNATIVES CONSIDERED:
[Options evaluated and why not recommended]

ATTACHMENTS:
[List of supporting documents]
```

## Recommendation Format

Recommendations must be written as formal motions that can be adopted verbatim:

### Single Recommendation
```
RECOMMENDATION:

THAT Staff Report [XX-XXX] regarding [subject] be received; and

THAT Council approve [specific action with clear parameters].
```

### Multiple Recommendations
```
RECOMMENDATIONS:

1. THAT Staff Report [XX-XXX] regarding [subject] be received.

2. THAT Council approve [first action].

3. THAT Council direct staff to [second action].

4. THAT the appropriate City Officials be authorized to take the
   necessary actions to give effect to this resolution.
```

## Financial Implications Section

Always include:

| Component | Details |
|-----------|---------|
| **Operating Budget Impact** | Annual cost/savings, which budget line |
| **Capital Budget Impact** | One-time costs, funding source |
| **Funding Source** | Tax levy, reserves, grants, user fees |
| **Multi-Year Implications** | Future year commitments |
| **Cost Recovery** | Revenue offsets if applicable |

### Example
```
FINANCIAL IMPLICATIONS:

Operating Budget Impact:
The proposed service expansion will require an additional $150,000
annually beginning in 2027. This represents a 2.1% increase to the
Transit operating budget. No provision exists in the current
approved budget.

Capital Budget Impact:
Two additional buses are required at a cost of $1.2 million.
This can be accommodated within the approved 2027 Capital Budget
for fleet replacement.

Funding Source:
Operating costs would require a tax levy increase or reallocation
from other programs. Capital costs are funded through the Transit
Fleet Reserve and ICIP grant funding (50% federal, 33% provincial).
```

## Strategic Alignment Section

Reference the City's Strategic Plan priorities:

```
STRATEGIC ALIGNMENT:

This initiative supports the following City of Barrie Strategic Plan
2022-2026 priorities:

Growing Our Economy
- Supports workforce mobility and access to employment areas

Building Strong Neighbourhoods
- Improves transit access in underserved communities

Fostering a Green Barrie
- Reduces single-occupant vehicle trips and emissions
```

## Writing Style Guide

1. **Third person, formal tone** - "Staff recommends" not "I recommend"
2. **Plain language** - Avoid jargon, define technical terms
3. **Active voice preferred** - "Council approved" not "was approved by Council"
4. **Specific dollar amounts** - "$1,234,567" not "approximately $1.2 million"
5. **Date references** - "June 15, 2026" not "next month"
6. **Reference previous decisions** - Cite report numbers and meeting dates

## Transit-Specific Considerations

When writing transit reports, include:

- **Ridership projections** with methodology
- **Service hours impact** (revenue hours added/removed)
- **Fleet implications** (peak vehicle requirement changes)
- **Accessibility compliance** (AODA requirements)
- **Regional coordination** (Metrolinx/GO Transit implications)

## Common Report Types for Transit

| Report Type | Key Sections to Emphasize |
|-------------|---------------------------|
| Service change | Ridership impact, public consultation results |
| Fare adjustment | Revenue projections, peer comparisons |
| Capital project | Procurement approach, project timeline |
| Strategic plan | Goals alignment, performance metrics |
| Annual update | KPI performance, year-over-year trends |

## Anti-Patterns to Avoid

- Burying the recommendation in analysis
- Vague financial implications ("cost neutral" without details)
- Missing stakeholder consultation section
- Recommendations that can't be adopted as motions
- Assuming Council knowledge of technical terms
