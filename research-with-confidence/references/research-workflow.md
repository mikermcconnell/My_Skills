# Research Workflow

## Clarify first

Confirm:
- topic
- time horizon
- depth
- source preferences if the user has any

## Suggested research angles

A good default set is:
- current developments
- primary facts or original data
- expert analysis
- criticism or counter-arguments
- implications for the user's situation

## Output structure

A strong research brief usually includes:
- executive summary
- findings table or bullet list
- confidence level for major claims
- what is still unclear
- relevance to the user's context
- source list
