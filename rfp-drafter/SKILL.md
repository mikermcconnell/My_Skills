# RFP Drafter for Municipal Procurement

Create professional procurement documents for consulting services following Ontario municipal standards.

## Document Types

| Type | When to Use |
|------|-------------|
| **RFP** (Request for Proposal) | Complex projects, qualitative evaluation matters |
| **RFQ** (Request for Quotation) | Price-focused, well-defined scope |
| **RFPQ** (Request for Pre-Qualification) | Shortlist vendors for later RFP |
| **RFS** (Request for Standing Offer) | Recurring services, on-call arrangements |

## Standard RFP Structure

```
1. INTRODUCTION
   1.1 Purpose
   1.2 Background
   1.3 Definitions

2. SCOPE OF WORK
   2.1 Project Objectives
   2.2 Deliverables
   2.3 Tasks and Activities
   2.4 Optional Services
   2.5 Out of Scope

3. PROPOSAL REQUIREMENTS
   3.1 Mandatory Requirements
   3.2 Technical Proposal
   3.3 Pricing Proposal
   3.4 Proposal Format

4. EVALUATION CRITERIA
   4.1 Evaluation Process
   4.2 Scoring Matrix
   4.3 Interview/Presentation (if applicable)

5. TERMS AND CONDITIONS
   5.1 Contract Terms
   5.2 Insurance Requirements
   5.3 City Policies

6. SUBMISSION INSTRUCTIONS
   6.1 Closing Date and Time
   6.2 Submission Method
   6.3 Contact Information

APPENDICES
A. Proposal Submission Form
B. Pricing Form
C. Reference Form
D. Background Documents
```

## Evaluation Criteria Template

### Standard Weighting (Adjust as needed)

| Criteria | Weight | Sub-Criteria |
|----------|--------|--------------|
| **Corporate Experience** | 20% | Similar projects, municipal experience |
| **Project Team** | 25% | Qualifications, availability, local presence |
| **Methodology** | 25% | Approach, understanding, innovation |
| **Work Plan** | 15% | Schedule, milestones, risk management |
| **Pricing** | 15% | Value for money (not lowest price) |

### Detailed Scoring Matrix

```markdown
## 4.2 Scoring Matrix

Proposals will be evaluated using the following criteria:

### Corporate Experience (20 points)
| Score | Criteria |
|-------|----------|
| 16-20 | Extensive experience with 5+ similar municipal transit projects |
| 11-15 | Solid experience with 3-4 similar projects |
| 6-10 | Some relevant experience with 1-2 projects |
| 0-5 | Limited or no directly relevant experience |

### Project Team (25 points)
| Score | Criteria |
|-------|----------|
| 21-25 | Senior team with deep transit expertise, excellent availability |
| 16-20 | Qualified team with relevant experience |
| 11-15 | Adequate qualifications, some concerns about capacity |
| 0-10 | Team lacks required expertise or availability |

### Methodology (25 points)
| Score | Criteria |
|-------|----------|
| 21-25 | Innovative approach demonstrating deep understanding |
| 16-20 | Sound methodology addressing all requirements |
| 11-15 | Adequate approach with minor gaps |
| 0-10 | Does not adequately address requirements |

### Work Plan (15 points)
| Score | Criteria |
|-------|----------|
| 13-15 | Realistic, detailed schedule with clear milestones |
| 9-12 | Reasonable schedule meeting key dates |
| 5-8 | Schedule has gaps or risks |
| 0-4 | Unrealistic or incomplete schedule |

### Pricing (15 points)
Calculated as: (Lowest Price ÷ Proponent's Price) × 15
```

## Transit-Specific Considerations

For transit planning RFPs, require:

### Mandatory Qualifications
- Previous municipal transit strategic planning experience
- CUTA or APTA membership/participation
- Understanding of Ontario transit funding (ICIP, Gas Tax)
- Knowledge of AODA compliance requirements

### Desirable Qualifications
- Experience with transit electrification planning
- On-demand/microtransit implementation experience
- PRESTO/fare integration expertise
- Ontario municipal transit experience

### Reference Projects
Request 3+ references from:
- Canadian municipal transit agencies
- Projects of similar scope (5-year strategic plans)
- Projects completed in last 5 years

## Timeline Guidelines

| RFP Type | Minimum Posting | Typical Award |
|----------|-----------------|---------------|
| Simple study | 2-3 weeks | 4-6 weeks |
| Strategic plan | 4-6 weeks | 8-12 weeks |
| Major project | 6-8 weeks | 12-16 weeks |

## Common Procurement Pitfalls

- Scope too vague (leads to change orders)
- Scope too prescriptive (limits innovation)
- Over-weighting price (quality suffers)
- Under-weighting price (overspending)
- Unrealistic timelines
- Missing mandatory requirements (AODA, insurance)
- Forgetting to request project manager dedication %
