# Strategic Plan Section Writer

Draft sections of the Barrie Transit 2027-2032 Strategic Plan using the approved structure, consistent style, and decision-oriented framing.

## Use these references

Read only what you need:
- `references/plan-section-templates.md` for the full table of contents and section templates
- `references/plan-writing-quality.md` for writing guidance, quality checks, and final review criteria

## Workflow

1. Identify which section or subsection the user needs.
2. Match the request to the approved plan structure before drafting.
3. Use a section-appropriate template instead of starting from a blank page.
4. Keep the writing practical, decision-oriented, and aligned with Barrie Transit context.
5. Where relevant, include targets, timelines, phasing, and implementation implications.

## Drafting rules

- Write like a real strategic plan, not a brainstorming memo.
- Tie recommendations to growth, service outcomes, affordability, and implementation reality.
- Prefer clear headings, tables, and phased actions over dense narrative.
- Use measurable statements where possible.
- Keep each section aligned with the likely audience: staff, leadership, council, and the public.

## Output expectation

Most sections should include:
- a clear section purpose
- a concise narrative
- structured tables or bullets where they improve clarity
- implementation or performance implications when relevant

## Gotchas

- Don't draft a section without first placing it in the approved plan structure.
- Don't mix aspirational language with no delivery path.
- Don't bury targets, timelines, or trade-offs in prose.
- Don't let every section sound the same; match the content to the section's job.
- Don't invent numbers or baseline values the source material does not support.
