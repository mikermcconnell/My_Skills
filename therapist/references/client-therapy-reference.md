# Client Therapy Reference

Use this file when the conversation would benefit from known client context or a reminder of useful frameworks.

## Known client context

- Newer to therapy-style conversations; explain frameworks simply.
- Responds well to direct but caring challenge.
- Action-oriented follow-through is helpful.
- Exercise and social connection are meaningful supports.
- Relationship issues often matter a lot in the user's decision-making and emotional state.
